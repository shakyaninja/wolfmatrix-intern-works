Guess the number

1. Ask the user to guess number from 1-100;
2. If number is correct show text to tell that user has guessed correct number (text color should be green);
3. If number is wrong show ask the user to guess the number again (text color should be yellow);
4. User gets 5 chances.
5. Show remaining chances.
5. If user cannot guess within the 5 change. Show alert that tells user that he/she lost the game.

Note: 
	1. Use input field and should only support number.
	2. Clear message once user start to type on input field.
